## What is Terraform?

Terraform is an open-source Infrastructure as Code (IaC) tool developed by HashiCorp that allows you to define, provision, and manage infrastructure using simple configuration files.

Instead of manually creating cloud resources, Terraform enables you to automate infrastructure creation in a declarative way.

 ## Why Terraform?

### Traditional infrastructure management:

 Manual setup (error-prone)
 Hard to scale
 Not reusable

### Terraform solves this by:

 Automating infrastructure
 Enabling version control
 Ensuring consistency
 Supporting multi-cloud (AWS, Azure, GCP)
